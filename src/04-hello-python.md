# 初识 Python

安装 Python 后，我们可以先启动 Python 的交互模式。

### Windows

按下 `SUPER`（“Win 键”），在开始菜单中输入 `python`，然后按回车。

### macOS/Linux

启动“终端”，输入 `python3`，然后按回车。

## 你好，世界

> 提示：可将输入法切换语言的快捷键修改为 <kbd>Ctrl</kbd>+<kbd>空格</kbd> 等组合，
> 
> 以避免与 <kbd>Shift</kbd>+<kbd>数字</kbd> 等特殊符号输入组合键发生冲突。

你现在应该能看到如下输出：

```text
Python 3.14.6 ...
Type "help", "copyright", "credits" or "license" for more information.
>>>
```

其中，`3.14.6` 是当前启动的 Python 版本。

现在输入 `print("Hello, world")`，然后按回车运行：

```python
>>> print("Hello, world")
Hello, world
```

## 输入算式

> 注意：`>>> ` 后为输入的计算表达式，其中额外的空格仅用于对齐和美观。
>
> `#` 及其后的内容称为“注释”，用于说明代码的作用、假设、思路等，输入时可忽略。

现在我们已经输出了 `Hello, world`，但这还不能体现 Python 的计算能力。

进入 Python 的交互模式后，我们可以尝试输入一些计算表达式。

```python
>>> 1 + 2 # 整数加法
3
>>> 1 - 10 # 整数减法
-9
>>> 21 // 4 # 整数除法
5
>>> 2 ** 3 # 整数乘方
8
>>> 2 ** 128 # 计算机位数限制不了我！
340282366920938463463374607431768211456
>>> 3 / 10 # 小数除法
0.3
>>> 5 / 1 # 反过来
5.0
>>> 15.5 * 2 # 仍然不是整数？
31.0
>>> 0.1 + 0.2 # 为什么？
0.30000000000000004
>>> 2 ** 0.5 # 小数次幂计算
1.4142135623730951
```

至此，我们已经拥有了一个方便使用、整数大小不受限制的计算器。
