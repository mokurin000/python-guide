# 《从 Python 入门软件开发》

## 目标人群

本书希望为因兴趣驱动而想要学习软件开发，或在日常工作中存在大量重复性任务的人群，提供可参考的学习路线与方向。

本书不假设读者此前拥有任何编程经验，并尽力避免让教学内容沦为术语的堆砌。

不过，良好的数学基础会对你很有帮助；学习英语也可以帮助你减少对 LLM 翻译或机器翻译的依赖。

## 开源社区

* [Github](https://github.com/)，Github 是代码托管平台，也可以为你提供协作开发的机会。
* [CodeBerg](https://codeberg.org/)，类似于 Github，但社区规模相对较小。
* [@py_zh_real](https://t.me/py_zh_real)，Telegram 中文 Python 交流群组。
* [@Python](https://t.me/Python)，英语 Python 群组，管理十分严格。

加入并参与开源社区，对软件开发的学习非常有帮助，尽管存在一定门槛。

不过，不要为了增加贡献数量而购买几个 Agent，然后到处提交纯 AI 生成的内容——这是对社区不负责任的行为。

截至目前，AI 仍然需要良好的 System Prompt 和正确的决策支持，才能真正发挥提升效率的作用。

Gitee、GitCode 等国产托管平台的审查相对严格，敏感词过滤机制较为原始，实名要求也较为繁琐；后者还曾出现未经授权搬运项目、冒名创建同名账户等争议，因此完全不推荐使用。

请远离 CSDN：它不仅广告众多、界面繁杂，而且软件开发领域的内容质量也普遍参差不齐。

### 浏览器

* [Chrome](https://www.google.cn/chrome/?standalone=1)，谷歌浏览器。
* [Firefox](https://www.firefox.com/zh-CN/)，火狐浏览器。

国产魔改浏览器对扩展安装的支持通常比较受限，它们大多基于 Chromium 进行深度修改。

## AI助手

- [DeepSeek](https://chat.deepseek.com/), 需手机号登录，是国产模型中能力相对最出众的。
- [ChatGPT](https://chatgpt.com/), 需要较干净的海外 IP ，模型的搜索能力、tool calling能力较好。
- [Gemini](https://gemini.google.com/), 逻辑能力表现相对欠佳，仅适合用于快速搜索资料链接。
