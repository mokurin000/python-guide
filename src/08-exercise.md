# 开始练习

有句名言叫“在实践中学习”，这对于软件开发的学习来说尤为重要。

如果只是生硬地啃书中提到的各种新概念，不仅容易让人感到枯燥乏味，也容易在理解抽象概念时产生偏差。

## 存储代码

在之前的代码示例中，我们一直直接在 Python 交互模式中输入代码并验证结果。

对于简短的代码片段来说，这种方式已经足够；但这样做既难以保存程序代码，也难以维护具有一定规模的程序。

Python 可以从文件中加载代码并运行。在开发环境中，我们不需要手动通过命令行运行 Python，只需打开本书源码中的 `run_tests.py`，然后点击“▷”即可。

## 第一个练习

> 我们将会使用 `print()` 函数进行输出。与交互模式不同，运行代码文件时，Python 不会自动输出每条语句的计算结果。
>
> 注意：`print()` 不仅可以输出 `str` 字符串，还可以输出数字、布尔值等各种类型的值。
