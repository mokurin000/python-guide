# 变量

在 Python 中，变量可以为某个值提供访问入口，并通过名称表达其含义。

我们已经认识了几种基础类型，但如果程序中直接使用大量字面常量，会使代码难以理解：

```python
>>> # 计算半径为 5 的圆面积
>>> 3.14159265 * 5**2
78.53981625
```

我们可以适当添加注释，但这并不是完美的解决方案。

```python
>>> pi = 3.14159265
>>> radius = 5
>>> pi * radius**2
78.53981625
```

通过使用具有明确含义的变量名，我们可以让复杂的数据处理流程更加易于阅读和理解。

## 变量名规则

Python 中的变量名，不止可以使用英语字母、下划线、阿拉伯数字。

主流语言的“文字”也可用作变量名命名：

```python
你好 = "Hello, World!" # 汉语
Grüße = "Hello, World!" # 德语
Привет = "Hello, World!" # 俄语
안녕하세요 = "Hello, World!" # 韩语
こんにちは = "Hello, World!" # 日语
```

不过，变量名不可以以数字开头。同时，作为 Python 语法一部分的“保留关键字”不可以用于定义变量。

## 类型提示

在前面的示例中，我们并没有在代码里标注变量（所引用的值）的数据类型。

Python 提供了 `type hinting`（类型提示）机制。不过，它与 `C` 语言中的类型声明不同，并不会对变量施加强制性的类型约束，而是作为面向开发者和工具的提示信息。

```python
integer: int = 5
real_number: float = 1.14514
hello: str = "Hello"

mismatched: int = 5.0 # 类型提示不会在运行时强制检查
```
