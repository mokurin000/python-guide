# 数据类型

> 本章节围绕主流平台上的 CPython 具体实现展开。

了解了 Python 交互模式中的表达式计算后，你可能已经发现：

* 有些算式的参与运算数虽然都没有小数点，但结果即使是整数，也会带有小数点。
* 有些算式的结果并不是完全精确或相等的（例如 `0.1 + 0.2` 不会得到 `0.3`）。

这里 Python 之所以会分别输出整数和 `整数.0`，是因为计算结果的 **数据类型** 不同。

定义数据类型有助于我们区分不同种类数据的处理方式。作为一个简单的开始，我们先忽略相对复杂的“对象”概念。

## 字面常量

字面常量顾名思义，就是直接表示其自身值的常量。

* `1`
* `3.14`
* `"这是一个字符串"`

## `int`

Python 中的 `int` 类型不同于大多数 C 系语言[^1]，它具有**无限精度**：

你可以直接计算非常大的整数，而不必担心因为数值过大导致计算结果出错。

> 数学中，我们最常用的进制是十进制。
> 
> 进制中的基数代表数每到多少进一位，每个进制单个位上表达的最大数字是进制基数减一。

Python 提供了多种语法来定义整数字面常量：

* `0b1111`，二进制
* `0o17`，八进制
* `15`，十进制
* `0xF`，十六进制

表达比较大的数字时，我们还可以适当插入“_”，它只用于对齐：`1_000_000_000`。

[^1]: C++、Java 等许多现代编程语言继承了 C 的语法体系。

## `float`

Python 中，`float` 表示计算机中的**浮点数**。

定义浮点数常量类似于数学中的写法：

- `.02`, 可以省略前后的 0 ，但是可读性欠佳
- `0.02`, 常见表示
- `2e-2`, 科学计数法表示

之所以称为“浮点数”，是因为现代计算机通常使用有限数量的二进制位来表示实数，而数值大小则通过调整指数来实现，相当于二进制小数点的位置可以“浮动”。

如果你希望更进一步理解数学细节，可见 [浮点算术：问题和限制](https://docs.python.org/zh-cn/3.14/tutorial/floatingpoint.html) 。

## `bool`

布尔值用于表示真假。这里的真假指的是逻辑表达式是否成立，而不是某个陈述是否符合事实。

布尔值只有两种：

* `True`：真
* `False`：假

逻辑表达式通常使用其他数值类型来表示，例如：

- `a > b`, a 大于 b
- `a <= b`, a 小于等于 b
- `a != b`, a 不等于 b
- `a == b`, a 等于 b

我们还可以使用逻辑运算符：

- `a > b and b != c`, `a > b` 与 `b != c` 均成立
- `not a == b`, `a == b` 不成立

## `str`

`str` 类型并不用于表示数字，而是用于表示文本内容。

* `"Hello, world"`，我们最常见的“你好，世界”示例所使用的文本。
* `"I'm there"`，你可以在由 `"` 包围的字符串中直接使用 `'`。
* `'"Hello", Bob said.'`，你也可以在由 `'` 包围的字符串中直接使用 `"`。
 
在其他情况下，我们需要使用额外的符号 `\` 来区分 `'`/`"` 究竟表示字符串的结束，还是字符串本身的内容。
```python
>>> print('"What\'s your problem?"，他说')
"What's your problem?"，他说
```

有时候，我们也需要表示多行文本。

> `...` 与 `>>>` 类似，都是 Python 交互模式中的提示符。
> 
> 它表示当前输入的内容尚未结束，需要继续输入后续内容。

```python
>>> print("""This is a multi-
... line text.""")
This is a multi-
line text.
```

Python 提供了 `"""` 和 `'''` 来表示这类多行文本。
